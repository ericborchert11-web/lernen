# LPR Schulungsraum · Prototyp

Interaktiver HTML-Prototyp des Online-Schulungsraums von **Leben Pflegen Reisen e.V.** —
Schulung „Psychiatrische Einzelbetreuung & Sitzwache".

**Status:** Demo-Mockup · kein Live-Betrieb · keine Datenspeicherung

**Live-Version:** https://USERNAME.github.io/lpr-schulungsraum/ _(nach Setup anpassen)_

---

## Was es tut

Eine komplette Schulung zum Durchklicken mit 5 Modulen und 24 Lektionen:

- Login-Screen mit Demo-Zugang
- Meine-Kurse-Übersicht mit Fortschrittsanzeige
- Kurs-Ansicht mit Sidebar-Navigation
- Lektionen mit interaktiven Praxis-Fragen
- Modul-Quizzes mit Auswertung
- Abschlussprüfung
- PDF-Zertifikat (via Browser-Druckfunktion)
- Handbuch-Ansicht zum Nachschlagen

## Tech

Ein einzelnes HTML-File, ca. 155 KB. Keine Abhängigkeiten außer Google Fonts.
Funktioniert offline, wenn man das File direkt im Browser öffnet.
